# **App Name**: NEULib Connect

## Core Features:

- User Authentication & Role Management: Securely manage user accounts for 'Admin' and 'Visitor' roles, allowing login and enforcing role-based access control.
- Visitor Check-in Form: Provide a simple interface for visitors to log their entry, capturing details such as reason for visit, college, and employee status.
- Personalized Welcome Message: Display a dynamic greeting, 'Welcome to NEU Library!', upon successful login or visitor check-in.
- Admin Visitor Dashboard: Admins can view aggregated visitor statistics, including visitor counts, displayed in informative cards for daily, weekly, or custom date ranges.
- Statistical Filtering Tools: Enable admins to filter visitor statistics based on parameters like reason for visit, associated college, and whether the visitor is an employee (teacher/staff).
- AI-Powered Visitor Insights: Generate intelligent summaries and highlight trends from visitor log data using an AI tool, providing admins with actionable insights on library usage patterns.

## Style Guidelines:

- Primary color: A sophisticated and deep indigo (#2e3a75), reflecting trust and academic gravitas against a light background.
- Background color: A very light, cool grey-blue (#F4F6FB), creating an airy and clean feel for the interface.
- Accent color: A vibrant yet refined blue (#3385cc), designed to draw attention to interactive elements and important information, providing good contrast.
- Headlines font: 'Alegreya' (serif), chosen for its elegant and intellectual feel, lending authority and sophistication to titles.
- Body font: 'Inter' (sans-serif), providing a modern, neutral, and highly readable choice for general text, forms, and statistical data.
- Utilize simple, clean line-art icons that complement the professional and academic tone of the application.
- Employ a modular card-based layout for the statistics dashboard to ensure data readability and ease of filtering.
